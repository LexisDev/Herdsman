# 🐑 Herdsman 2D

![Herdsman 2D Screenshot](./public/assets//screenshot.png)

**Herdsman 2D** — это минималистичная 2D-игра, в которой ты управляешь пастухом и собираешь разбросанных животных, чтобы доставить их в загон.

---

## 🎮 Gameplay

- Управляй героем кликами по полю
- Подбирай животных — они начинают следовать за тобой
- Собирай цепочку
- Доставляй животных в загон (жёлтая зона)
- Получай очки за каждое доставленное животное
- После доставки всех животных начинается новый цикл
